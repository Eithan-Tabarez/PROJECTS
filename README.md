# Mi Portafolio Personal

## Descripción

Este proyecto consiste en el desarrollo de una página web personal en la que presento información sobre mí, mis habilidades y algunos de los proyectos que he realizado. El objetivo es crear un sitio web sencillo, atractivo y fácil de navegar que funcione como una carta de presentación en el ámbito del desarrollo web.

## Objetivos

* Presentar información personal de forma clara.
* Mostrar algunos de mis proyectos más relevantes.
* Practicar el uso de HTML5 y la estructura semántica de una página web.
* Aplicar buenas prácticas en el desarrollo web.

## Estructura de la página

La página contará con las siguientes secciones:

* **Inicio:** Presentación y bienvenida.
* **Sobre mí:** Información personal, intereses y una fotografía.
* **Proyectos:** Lista de proyectos con enlaces a sus respectivos repositorios o demostraciones.
* **Contacto:** Información para comunicarse conmigo.

## Tecnologías utilizadas

* HTML5
* CSS3
* Git
* GitHub

## Características

* Diseño limpio y responsive.
* Navegación mediante un menú.
* Uso de etiquetas semánticas de HTML.
* Enlaces externos a GitHub.
* Imágenes con texto alternativo para mejorar la accesibilidad.
